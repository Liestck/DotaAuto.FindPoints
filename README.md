# DotaAuto Coordinate Finder

Вспомогательная утилита проекта DotaAuto для быстрого определения координат областей на скриншоте Dota 2 с возможностью экспорта выбранной области, увеличенной лупой и точной подстройкой координат.

![Демонстрация](./demo.gif)

---

## Возможности

- Выбор двух точек на скриншоте для выделения прямоугольной области.
- Перетаскивание точек мышью для точной корректировки.
- Лупа при перетаскивании точки с сеткой и красной точкой в центре.
- Отображение текущих координат точки под лупой.
- Перемещение точек с помощью клавиш стрелок на 1 пиксель для тонкой подстройки.
- Копирование координат точек в буфер обмена.
- Экспорт выбранной области в PNG с именем в формате:
  ```
  dd.mm.yy_left.top_right.bottom.png
  ```
- Автоматическое масштабирование изображения под окно программы.

---

## Использование

1. **Выставление точек:**
   - Нажмите кнопку `Выставить координаты`.
   - **ПКМ** (правая кнопка мыши) — установка первой точки (`Point1`) и второй точки (`Point2`).
   - После установки точек можно перетаскивать их **ЛКМ** для точной корректировки.

2. **Лупа и координаты:**
   - При перетаскивании точки появляется увеличенная лупа (радиус 30 px, масштаб ×3).
   - В лупе отображается сетка и красная точка в центре.
   - Под лупой показываются текущие координаты точки в оригинальном разрешении скриншота.

3. **Перемещение стрелками:**
   - Если ЛКМ зажат и выбрана точка, используйте стрелки **↑↓←→** для смещения точки на 1 пиксель в соответствующем направлении.

4. **Копирование координат:**
   - После выставления обеих точек доступна кнопка `Скопировать`.
   - Координаты копируются в буфер обмена в формате:
     ```
     Point1: (x1, y1) | Point2: (x2, y2)
     ```

5. **Экспорт выделенной области:**
   - Нажмите кнопку `Экспорт`.
   - Файл сохранится в папку `export` рядом с программой.
   - Имя файла формируется как:
     ```
     dd.mm.yy_left.top_right.bottom.png
     ```
     например: `18.01.26_540.262_1322.328.png`.

---

## Настройки

- Размер скриншота Dota 2 (`DEFAULT_SCREEN_WIDTH` и `DEFAULT_SCREEN_HEIGHT`).
- Размер окна программы (`DISPLAY_WIDTH`, `DISPLAY_HEIGHT`).
- Цвета и радиусы точек/прямоугольника.
- Радиус и масштаб лупы (`ZOOM_RADIUS` и `ZOOM_SCALE`).

---

## Примечания

- Программа работает с активным окном Dota 2.
- Убедитесь, что Dota 2 открыта и окно не свернуто перед запуском.
- Лупа появляется только при **перемещении точки**, чтобы избежать пустого окна при клике.
